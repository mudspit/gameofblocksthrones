# Game of Blocks Thrones

A Minecraft-style Game of Thrones RPG that runs entirely in the browser — no install, no build step. Pure Three.js voxels.

**Play online:** https://mudspit.github.io/gameofblocksthrones/

## The Story So Far

You are a hedge knight who just inherited Mudford Keep — walls, debts, and all. Rise through two acts:

- **Act I — The Hedge Knight**: thin the wolf packs, forge castle steel, and slay Bandit King Rorge to earn your lordship
- **Act II — Fire and Blood**: hatch a dragon from Rorge's plunder, recruit allies, survive the Vale's raiders, arm yourself with dragonglass, hunt White Walkers in the dark, and slay the Wild Dragon from dragonback
- **Act III** — coming soon: the march on the capital

## Features

- 192×192 procedurally meshed voxel world: keep, village, forests, bandit camp, frozen cairns, dragon lair
- 5 weapons: sword (upgrades to Valyrian steel), hunting bow, dragonglass dagger, warhammer, Myrish crossbow
- Raise, feed, and **ride a dragon** — W flies where you look, LMB breathes fire
- Ride a horse (E to mount, H to whistle)
- White Walkers and wights stalk every night — steel can't touch the pale ones
- Allies who fight beside you: Snow the direwolf and Ser Bryn
- Quest chain with dialogue, a world map (M), clickable mission beacons, XP/levels, loot drops, medical kits
- **Accounts & saves** — create a named account on the title screen; progress autosaves in your browser

## Controls

| Key | Action |
|---|---|
| WASD / Mouse | Move / look |
| 1–5 | Switch weapons |
| LMB / RMB | Attack, chop / place block |
| E | Talk, ride, dismount, feed |
| Q / F | Bandage (+30 hp) / Maester's Kit (full heal) |
| H | Whistle your horse |
| M | World map |
| Space / Shift | Jump / sprint |

## Run locally

```
node server.js
```

Then open http://localhost:4173. (Any static file server works.)
